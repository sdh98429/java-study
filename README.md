#### JDK 환경 변수 설정

> javac 명령은 자바 파일을 컴파일하여 실행하기 위해서 필요한 명령
>
> JDK 설치 후 환경 변수 설정을 제대로 하면 CMD(명령프롬프트)에서 javac 입력시 정상적으로 컴파일 관련 옵션이 출력됨
>
> 환경 변수 서정이 되어있지 않으면 javac 기능이 인지 안되기 때문에 환경변수 설정에서 경로 지정이 중요
